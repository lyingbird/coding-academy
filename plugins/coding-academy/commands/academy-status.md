# /academy:status

Shows the current companion profile and recent progress.

Run this command first:

- `node "$CLAUDE_PLUGIN_ROOT/bin/academy-status.js"`

Then summarize:

- current hero state
- active or recent battle mood
- level, streak, and souvenirs
- whether the hero looks idle, hurt, winning, or ready for a new quest
